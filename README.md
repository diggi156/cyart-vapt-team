🔐 CyArt VAPT Internship

📌 Overview

This repository contains weekly work completed during the Vulnerability Assessment and Penetration Testing (VAPT) Internship under the CyArt VAPT Team.

Each week focuses on different aspects of cybersecurity, including theoretical learning, practical implementation, and reporting.
