Practical Application – Week 3

📌 Overview

This document describes the practical activities performed during Week 3 of the VAPT internship, including exploitation, web application testing, post-exploitation, and reporting.

---

🔹 1. Advanced Exploitation Lab

Tools Used

- Metasploit Framework
- Exploit-DB
- Kali Linux

Activities Performed

- Configured exploit modules in Metasploit
- Selected appropriate payloads
- Executed exploit on target system (Metasploitable2)

Result

- Successful exploitation achieved
- Access to target system obtained

---

🔹 2. Web Application Testing Lab

Tools Used

- DVWA (Damn Vulnerable Web Application)
- Burp Suite
- Browser (Manual Testing)

Activities Performed

- Tested for Cross-Site Scripting (XSS)
- Tested for SQL Injection
- Intercepted and modified requests using Burp Suite

Result

- XSS vulnerability successfully demonstrated
- SQL Injection vulnerability identified

---

🔹 3. Reporting Practice

Activities Performed

- Documented vulnerabilities in structured format
- Created findings table with severity and remediation
- Prepared basic report layout (Executive Summary + Findings)

Result

- Clear and structured reporting achieved

---

🔹 4. Post-Exploitation and Evidence Collection

Tools Used

- Meterpreter
- Wireshark

Activities Performed

- Established session with target system
- Executed commands ("whoami", "sysinfo")
- Attempted privilege escalation
- Captured network traffic using Wireshark
- Collected screenshots as proof

Result

- System access verified
- Evidence successfully collected

---

🔹 5. Capstone Project (Full VAPT Cycle)

Activities Performed

- Performed complete VAPT cycle:
  - Scanning
  - Exploitation
  - Post-exploitation
  - Reporting

Result

- Successfully demonstrated end-to-end penetration testing workflow

---

🎯 Conclusion

The practical implementation provided hands-on experience in exploiting vulnerabilities, performing web application testing, and documenting findings. This strengthened the understanding of real-world VAPT processes.
