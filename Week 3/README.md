🔐 Week 3 – Advanced VAPT

📌 Overview

This week focuses on advanced concepts of Vulnerability Assessment and Penetration Testing (VAPT), including exploitation techniques, web application security testing, post-exploitation, and reporting.

All activities were performed in a controlled lab environment using vulnerable machines and security tools.

---

🧭 Methodology

The following VAPT phases were performed:

1. Reconnaissance
2. Scanning & Enumeration
3. Exploitation
4. Post-Exploitation
5. Evidence Collection & Reporting

---

⚙️ Work Completed

🔹 1. Advanced Exploitation

- Configured exploits using Metasploit
- Executed exploits on target machine
- Established Meterpreter session

🔹 2. Web Application Testing

- Performed XSS (Cross-Site Scripting)
- Tested for SQL Injection
- Used DVWA for vulnerability testing

🔹 3. Post-Exploitation

- Accessed target system
- Performed basic privilege escalation
- Verified system-level access

🔹 4. Evidence Collection

- Captured screenshots of all attack stages
- Collected logs and session details
- Used Wireshark for traffic analysis
