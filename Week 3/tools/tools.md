Tools Used – Week 3

📌 Overview

This document lists the tools and technologies used during the Week 3 VAPT activities.

---

🛠️ Tools & Technologies

🔹 Kali Linux

- Primary operating system for penetration testing
- Used for running all security tools

---

🔹 Metasploit Framework

- Used for exploitation
- Configured and executed exploits
- Established Meterpreter sessions

---

🔹 DVWA (Damn Vulnerable Web Application)

- Used for web application testing
- Practiced vulnerabilities like XSS and SQL Injection

---

🔹 Burp Suite

- Intercepted HTTP requests
- Modified requests for testing vulnerabilities

---

🔹 Wireshark

- Captured network traffic
- Used for evidence collection

---

🔹 Nmap

- Performed network scanning
- Identified open ports and services

---

🔹 MySQL

- Used as backend database for DVWA
- Tested database-related vulnerabilities

---

🎯 Conclusion

These tools enabled effective execution of VAPT activities including scanning, exploitation, testing, and reporting.
