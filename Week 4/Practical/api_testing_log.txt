## [2] Broken Object Level Authorization (BOLA)

Steps:

* Captured request in Burp Suite
* Modified parameter:
  id=1 → id=2

Result:

* Accessed data of another user
* Unauthorized access confirmed

Severity:

* Critical

* Modified input fields in intercepted requests
* Tested different payloads

## [4] Authentication Testing

Steps:

* Intercepted login request

Result:

* Weak authentication mechanism observed

## [5] Rate Limiting Test

Result:

* No rate limiting detected
* System vulnerable to brute force
